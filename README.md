## Using pnpm

This project uses pnpm. Common commands:

- Install deps: `pnpm install`
- Start dev server: `pnpm dev`
- Build: `pnpm build`
- Test: `pnpm test`
- Lint: `pnpm lint`
- Preview: `pnpm preview`
